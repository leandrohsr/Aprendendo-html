Citações.

Para fazer uma citação, será possível de duas formas: Uma na mesma linha utilizando a tag <q> que significa quote (citação em inglês) e outra em um bloco utilizando a tag <blockquote cite="link da citação">, onde este cite dentro do blockquote não é obrigatório no código.

Criação de listas.

Para criar uma lista ordenada é utilizada a tag <ol> (que sigifica ordered list) e para cada item da lista é colocada a tag <li> que significa list item(item da lista). O type é colocado de forma opcional para mudar o tipo do marcador que vai ordenar a lista (números, letras ou algarismos romanos).

O rel pode ter vários atributos, entre eles:

  • next = indica que o link é para a próxima parte do documento atual.
  • prev = indica que o link é para a parte anterior ao documento atual.
  • author = indica que é um link para o site do autor do artigo atual.
  • external = indica que é um link para outro site que não faz parte do site construído.
  • nofollow = indica para o mecanismo de busca que não tem nada a ver um site com o outro.

As vezes um site a ser conectado está em outro idioma, e para indicar uma tradução para o navegador, coloca-se a tag <hreflan="en">. Direcionar para W3Schools.

Imagem Dinâmica.

Aumente e diminua a lateral da janela do seu navegador.

Este tipo de tag, é uma forma de adaptar o tamanho da imagem de acordo com o dispositivo que o usuário está navegando, seja ele um computador desktop, um tablet ou um smartphone.

Dentro da tag <picture> é colocado um <img> com a imagem principal. Logo depois acima é colocado a tag <source:media:type>, onde terá o media="(max-width: px)" para colocar a partir de qual tamanho será mudado a imagem; srcset="diretório da nova imagem com o arquivo que será mudado"; e o type="media type com o formato do arquivo".
